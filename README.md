## ⭐ Introduction

AI-Powered Placement Management System is a full-stack web application developed to streamline campus recruitment and placement activities. The platform enables placement officers to manage students, companies, interview schedules, recruitment results, and placement records through a centralized dashboard.

The system reduces manual effort in handling placement drives and provides an efficient workflow for managing recruitment processes.

### ✨ Features

* Employee Registration & Authentication
* Student Management
* Company Management
* Placement Drive Management
* Interview Scheduling
* Placement Result Tracking
* Resume Management
* CSV Report Generation
* Profile Management
* Responsive User Interface
* Secure Session Authentication
* Google OAuth Integration

### 🤖 Future AI Enhancements
